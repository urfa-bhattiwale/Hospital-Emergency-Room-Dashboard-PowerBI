## 📌 Project Overview

This project is an end-to-end Power BI dashboard developed to analyze Emergency Room (ER) operations and transform raw healthcare data into actionable business insights.

The dashboard helps hospital administrators monitor patient admissions, waiting times, satisfaction scores, referrals, and demographic trends, enabling better operational planning and data-driven decision-making.


## 🎯 Business Problem

Emergency departments handle a large volume of patients every day. Without centralized reporting, it becomes difficult to monitor hospital performance, identify operational bottlenecks, and allocate resources efficiently.

This dashboard provides an interactive solution to improve visibility into Emergency Room operations.

- High-demand departments receive more referrals.
- Wait time directly impacts patient satisfaction.
